Minimal Next.js + Tailwind + Three.js starter scaffold tailored for a 3D homepage.

How to run

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000

Notes
- The hero 3D scene lives in `components/Hero3D.tsx` (uses raw Three.js).
- Global font is Cascadia Code imported via CDN in `styles/globals.css`.
- Tailwind is configured in `tailwind.config.js`.

This README contains inline instructions in files (comments) about how the code works and where to tweak visuals/animation.
